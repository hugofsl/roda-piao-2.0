== Description ==

Daisy Titles is a simple WordPress plugin that allows you to customize how your page and post titles appear on the front-end of your website. 

With this plugin, you can control over the visibility of titles of pages and single posts on your WordPress website. Whether you want to show or hide titles, ‘Daisy Titles’ has got you covered.

Visibility Features:

* hide post titles
* hide page titles
* hide all titles
* hide specific post/page titles

Styling Features:

* Change the color of post titles
* Adjust the font size of titles
* Select from several common web fonts
* Choose which post types to apply the styling to (posts, pages, etc.)

The plugin includes a user-friendly settings panel in the WordPress admin area and follows all WordPress coding standards for security and performance.

== Installation ==

1. Upload the `daisy-titles` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Settings → Daisy Titles to configure your title styles

== Frequently Asked Questions ==

= Which post types can I style with this plugin? =
The plugin works with all public post types including posts, pages, and custom post types. You can select which ones to apply the styling to in the settings.

= Can I use custom fonts? =
Currently, the plugin offers a selection of common web-safe fonts. For custom fonts, you would need to add CSS to your theme or use a custom CSS plugin.

= Will this affect my SEO? =
No, the plugin only adds inline styles to your titles and doesn't modify the actual content or HTML structure that search engines see.

= Can I use different styles for different post types? =
Not in the current version. The plugin applies the same styling to all selected post types.
